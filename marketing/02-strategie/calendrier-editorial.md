## Process de production

1. **Avant rédaction** : mot-clé principal validé + intention SERP vérifiée (Google → top 5) + plan H2/H3 rédigé
2. **Rédaction** : par moi (agent), 80 % auto-écrit, 20 % validé/relu par toi
3. **Optimisation** : title 45-58 car. + description 130-155 car. + JSON-LD + maillage interne (au moins 3 liens sortants vers piliers)
4. **Publication** : push commit avec tag `seo: <slug>`, déploiement Coolify
5. **Promotion** : si article majeur → 1 post LinkedIn + 1 X thread dans les 48 h
6. **Suivi** : vérifier indexation GSC sous 7 jours, position SERP sous 30 jours
